* 이 배치파일은 개발된 지 오래되어 번거로운 과정을 거쳐야만 정상적으로 실행시킬 수 있습니다.
* 그러나 걱정하지 마십시오. 친절한 가이드가 여기 있습니다.

1. 압축폴더을 다운로드 후 바탕화면으로 옮기십시오. **중요**
2. 압축해제 후 이름을 `달을 보지 마십시오`로 바꾸십시오. (띄어쓰기까지 정확히 하여 생성해야 합니다) **중요**
3. 폴더을 열고 'A.vbs' 파일을 우클릭 한 후 (추가옵션 표시) - '편집'을 누르십시오. **중요**
4. 창이 하나 열리고 그곳엔 사용자 이름을 적는 곳이 있을 것 입니다. **중요**
5. 윈도우키+E를 누르고 C:/사용자(Users)로 가서 본인의 사용자 이름을 찾은 후 그곳에 적으십시오. **중요**
6. '시작.bat' 파일을 우클릭 한 후 (추가옵션 표시) - '편집'을 누르십시오. **중요**
7. 스크롤을 내리면 그곳에도 사용자 이름을 적는 곳이 있을 것 입니다. 5.에서 찾은 사용자 이름을 적어주십시오. **중요**
8. 윈도우키+S를 누르고 UAC(사용자 계정 컨트롤)를 검색하여 가장 낮은 단계로 설정하십시오. (끄십시오) **중요**
모두 완료했다면 준비는 끝났습니다.

'시작.bat'을 눌러 시작하십시오.

상황을 끝내고 싶다면 PC를 로그아웃 하십시오.

사용을 마친 후 보안을 위해 UAC를 3단계 이상으로 돌려 놓는 걸 잊지 마십시오.
